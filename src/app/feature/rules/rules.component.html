Le concept

Regexword est un jeu pour les amoureux de RegEx et de jeux de puzzle. Le challenge que vous propose regexword est de trouver quotidiennement un mot en vous donnant pour seul indice des regex qui se dévoilent dans le temps tout au long de la journée.

Les règles

  • Vous avez le droit à 6 essaies
  • Votre champs de saisie est la premier ligne bleu
  • Vous pouvez utiliser toutes les lettres de l'alphabet pour taper un mot
  • Utiliser la touche 'Backspace' pour effacer la/les dernière(s) lettre tapée(s)
  • Appuyez sur la touche 'Enter' valider une réponse.
  • Toutes les lettres doivent être saisies pour pouvoir valider la ligne
  • Si la ligne est rouge, n'avez pas trouvé le bon
  • Si la ligne est verte, bravo vous avez gagné pour aujourd'hui
